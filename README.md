# Vecode
An open source, in browser AI coding agent that is simple to use.


### What is Vecode?
Built with GLM 5.2 in OpenCode, Vecode is the easiest to use AI coding agent. it supports a free tier with Deepseek v4 Flash, or you can choose to bring your own key. It's Simple to use for beginners, works entirely within your browser with a virtual file system, and doesn't require installations or complex setups.

### A Stable Agent, maximum efficency.
Vecode is made in just a single index.html file. And that file isn't so big. However, Vecode is still extremley powerful, especially at things such as design. In testing, indeed, with MiMo V2.5, a very small model that is quite bad at coding, with Vecode, MiMo V2.5 delivered an almost perfect output with amazing design, compared to MiMo v2.5 on OpenCode, which has a less appealing design.
