How to use Vecode

What is Vecode?

Vecode is an easy-to-use AI coding agent that runs entirely inside your browser. You describe what you want to build in plain English, and the agent writes the code, tests it, and shows you a live preview — all without installing anything.

It uses a ReAct (Reason + Act) loop: the AI thinks about your request, calls tools to write files and run code, observes the results, and iterates until the task is done.

Getting started — BYOK (Bring Your Own Key)

Vecode needs an API key from an OpenAI-compatible provider. This is called BYOK — you bring your own key, so you control which model runs and what you pay. Your key is stored only in your browser's localStorage and is sent only to the endpoint you configure.

  1. Click Settings in the top bar.
  2. Pick a provider preset (OpenAI, OpenRouter, Together, Gemini, or local Ollama) or enter a custom endpoint URL.
  3. Paste your API key into the API Key field. Keys look like sk-... for OpenAI, sk-or-... for OpenRouter, etc.
  4. Enter or confirm the Model Name (e.g. gpt-4o, anthropic/claude-3.5-sonnet, llama3).
  5. Click Save Settings. The green dot in the logo means you're connected.
Where to get keys: OpenAI keys come from platform.openai.com/api-keys. OpenRouter keys from openrouter.ai/keys. For local models, install Ollama and run ollama serve — no key needed, just set the endpoint to http://localhost:11434/v1.

Using the agent

  • Type a request in the chat box, e.g. "Build a todo app with local storage and dark mode".
  • Press Enter to send, Shift+Enter for a newline.
  • The agent writes files to the VFS, runs JS to test, and renders a live preview. Watch the Console pane for logs.
  • Click Stop to abort a running generation.
  • Click any file in the file tree to view its code with syntax highlighting.

Export & import

  • Export → .zip: download all files as a zip archive (built-in ZIP encoder, no libraries).
  • Export → bundled HTML: one self-contained index.html with all CSS/JS inlined.
  • Export → JSON: save the whole VFS as a .json you can re-import later.
  • Import: load a previously exported JSON to restore a project.

Tips for best results

  • Be specific: "a pomodoro timer with a circular progress ring and sound alerts" beats "a timer".
  • The agent works best with models that support tool/function calling (gpt-4o, claude-3.5-sonnet, llama-3.1-70b).
  • You can edit the System Prompt in Settings to change the agent's persona or design philosophy.
  • Switch themes in Settings — Carbon, Slate, Solarized, or Tokyo Night.
  • Use New to wipe the VFS and start fresh.
